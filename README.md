# Bear's Anime

Персональная аниме-библиотека на Flask.

## Структура проекта

- `flask_app.py` — Flask-приложение
- `static/` — CSS и данные (`anime.json`)
- `templates/` — HTML-шаблоны
- `requirements.txt` — зависимости для PythonAnywhere
- `README.md` — описание проекта

## Как запустить локально

```bash
pip install -r requirements.txt
python flask_app.py
